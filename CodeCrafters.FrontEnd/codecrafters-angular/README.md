## CodeCrafters Frontend (Angular Placeholder)

This folder is a placeholder for the Angular frontend application.

- The final Angular app should be generated here using the Angular CLI, e.g.:
  - `cd CodeCrafters.FrontEnd/codecrafters-angular`
  - `ng new codecrafters-angular --routing --style=scss`
- Use a **module-based architecture** with the following structure:
  - `src/app/core`
  - `src/app/shared`
  - `src/app/layout`
  - `src/app/features`

The .NET Aspire AppHost is expected to orchestrate the frontend dev server once the Angular project is created and wired up.
